# Sello Madrid Excelente — materiales gráficos oficiales

Materiales oficiales del sello **Madrid Excelente** (empresa certificada) para incorporarlo a documentos
y comunicaciones a clientes y proveedores. Se versionan aquí para que estén disponibles en cualquier
entorno de desarrollo sin depender de descargas locales.

Issue maestra: **#245** · Implementación en NestoAPI: **#244**.

## Archivos

| Archivo | Uso |
|---|---|
| `sello-reducida.png` | **Versión reducida.** La que se usa en código (PDFs QuestPDF, firmas de correo, web, app). Es la indicada para soportes digitales y tamaños pequeños. |
| `sello-ampliada-baja.png` | Versión ampliada (baja resolución), para previsualización. |
| `sello-ampliada-versiones-baja.pdf` | Versión ampliada con todas sus variantes (baja resolución). |
| `arte-final-imprenta-ampliada.pdf` | Arte final para **imprenta** (adhesivos, cartelería, packaging). |
| `manual-uso-2025.pdf` | Manual oficial de comunicación y uso del sello (2025). |

> Los vídeos del sello (horizontal/vertical, normal/bucle) no se incluyen por tamaño; están en el pack
> original `graficas.zip` / OneDrive.

## Normas de uso (resumen del manual oficial 2025)

- **Dos versiones:** *ampliada* para soportes físicos grandes; **reducida** para digital y tamaños pequeños.
  En el software usamos siempre la **reducida**.
- **Carácter accesorio:** el sello **nunca** sustituye ni supera en tamaño al logo de la empresa; se coloca
  **a la derecha o debajo** del logo principal.
- **Prohibido** modificar el sello, sus proporciones o sus colores, y usar el logotipo suelto sin el resto
  de elementos.
- **Tamaños mínimos** (legibilidad del QR): reducida **30 × 14,23 mm**; ampliada otros materiales
  **40 × 34,12 mm**.
- **Colores oficiales:** rojo PANTONE 032 (#FF0000), azul PANTONE 2186 (#004986), negro (#000000),
  blanco (#FFFFFF).
